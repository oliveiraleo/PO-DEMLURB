-> Modulo alternativo de leitura de dados de um arquivo CSV

- Para utilizar esse modulo, basta compilar:

g++ main.cpp -o leitura.exe

- E executar usando:

./leitura.exe

- Saida de exemplo:

0 no_inicial,no_final,custo,endereco
1 0,19,78.92,Rua A 12 – 158
2 0,1,131.91,Rua D 106 – 280
[...]
38 23,21,975.58,Av. B 57 / Av. A 1580

A leitura de 39 linhas foi realizada com sucesso!
